# Connection

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/system/src/types/general.ts#L24)

The `Connection` type is the basic minimal description of an [`Edge`](./Edge)
between two nodes. The [`addEdge`](../utils/addEdge) util can be used to upgrade
a `Connection` to an [`Edge`](./Edge).

```ts
export type Connection = {
  source: string | null;
  target: string | null;
  sourceHandle: string | null;
  targetHandle: string | null;
};
```
