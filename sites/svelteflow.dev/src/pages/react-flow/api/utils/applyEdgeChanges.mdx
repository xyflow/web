# applyEdgeChanges()

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/react/src/utils/changes.ts#L122)

Various events on the [`<ReactFlow />`](../ReactFlow) component can produce an
[`EdgeChange`](../types/EdgeChange) that describes how to update the edges of your
flow in some way. If you don't need any custom behaviour, this util can be used to
take an array of these changes and apply them to your edges.

```js
import { useState, useCallback } from 'react';
import ReactFlow, { applyEdgeChanges } from '@xyflow/react';

export default function Flow() {
  const [nodes, setNodes] = useState([]);
  const [edges, setEdges] = useState([]);
  const onEdgesChange = useCallback(
    (changes) => {
      setEdges((oldEdges) => applyEdgeChanges(changes, oldEdges));
    },
    [setEdges]
  );

  return (
    <ReactFLow nodes={nodes} edges={edges} onEdgesChange={onEdgesChange} />
  );
}
```
