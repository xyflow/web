# DefaultEdgeOptions&lt;T>

Many properties on an [`Edge`](/api-reference/types/edge) are optional. When a new edge is created,
the properties that are not provided will be filled in with the default values
passed to the `defaultEdgeOptions` prop of the [`<SvelteFlow />`](/api-reference/svelte-flow#defaultedgeoptions)
component.

```ts
export type DefaultEdgeOptions<T> = {
  type?: string;
  animated?: boolean;
  hidden?: boolean;
  deletable?: boolean;
  selectable?: boolean;
  data?: T;
  selected?: boolean;
  markerStart?: string | EdgeMarker;
  markerEnd?: string | EdgeMarker;
  zIndex?: number;
  ariaLabel?: string;
  interactionWidth?: number;
};
```
