# FitViewOptions

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/react/src/types/general.ts#L43)

When calling `fitView` these options can be used to customize the behaviour. For example, the `duration` option can be used to transform the viewport smoothly over a given amount of time.

```ts
export type FitViewOptions = {
  padding?: number;
  includeHiddenNodes?: boolean;
  minZoom?: number;
  maxZoom?: number;
  duration?: number;
  nodes?: (Partial<Node> & { id: Node['id'] })[];
};
```
