# EdgeMarker

You can customize the built-in edge markers with the `edgeMarker` [Edge](/api-reference/types/Edge) prop.

```ts
export type EdgeMarker = {
  type: MarkerType;
  color?: string;
  width?: number;
  height?: number;
  markerUnits?: string;
  orient?: string;
  strokeWidth?: number;
};
```
