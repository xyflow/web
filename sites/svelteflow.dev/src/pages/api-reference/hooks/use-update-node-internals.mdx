# useUpdateNodeInternals

[Source on GitHub](https://github.com/wbkd/react-flow/blob/78169c992bf95a5dbc9bccbdc2a07725a307e2c3/packages/svelte/src/lib/hooks/useUpdateNodeInternals.ts/#L6)

When you programmatically add or remove handles to a node or update a node's
handle position, you need to inform Svelte Flow about it by using this hook. This
will update the internal dimensions of the node and properly reposition handles
on the canvas if necessary.

```svelte
<script lang="ts">
  import { Handle, useUpdateNodeInternals } from '@xyflow/svelte';

  const updateNodeInternals = useUpdateNodeInternals();
</script>
```
