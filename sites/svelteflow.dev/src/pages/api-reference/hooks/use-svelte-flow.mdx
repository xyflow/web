# useSvelteFlow

This hook returns functions to update the viewport, transform coordinates or get node intersections for example.

```svelte
<script lang="ts">
  import { useSvelteFlow } from '@xyflow/svelte';

  const { screenToFlowCoordinate, zoomIn } = useSvelteFlow();
</script>
```
