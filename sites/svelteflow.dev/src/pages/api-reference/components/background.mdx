# &lt;Background />

[Source on GitHub](https://github.com/wbkd/react-flow/blob/78169c992bf95a5dbc9bccbdc2a07725a307e2c3/packages/svelte/src/lib/plugins/Background/Background.svelte)

The `<Background />` component makes it convenient to render different types of
backgrounds common in node-based UIs. It comes with three variants: `lines`,
`dots` and `cross`.

```svelte
<script lang="ts">
  import { writable } from 'svelte/store';
  import { SvelteFlow, Background, BackgroundVariant } from '@xyflow/svelte';

  const nodes = writable([]);
  const edges = writable([]);
</script>

<SvelteFlow nodes={nodes} edges={edges}>
  <Background color="#ccc" variant={BackgroundVariant.Dots} />
</SvelteFlow>
```
