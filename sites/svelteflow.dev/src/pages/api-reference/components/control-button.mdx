# &lt;ControlButton />

[Source on GitHub](https://github.com/wbkd/react-flow/blob/78169c992bf95a5dbc9bccbdc2a07725a307e2c3/packages/svelte/src/lib/plugins/Controls/ControlButton.svelte)

The `<ControlButton />` component is used to render custom/ additional buttons
for the `<Controls />` component.

```svelte
<script lang="ts">
  import { Controls, ControlButton } from '@xyflow/svelte';
</script>

<Controls>
  <ControlButton on:click={() => console.log('⚡️')}>
    ⚡️
  </ControlButton>
</Controls>
```
