Lots of improvements in this release! We added a new `EdgeToolbar` component, it's now
possible to start a selection above a node and child nodes of different parents won't
overlap anymore:

### Minor Changes

- [#5544](https://github.com/xyflow/xyflow/pull/5544) Add `EdgeToolbar` component
- [#5550](https://github.com/xyflow/xyflow/pull/5550) Prevent child nodes of different
  parents from overlapping
- [#5551](https://github.com/xyflow/xyflow/pull/5551) Allow to start a selection above a
  node
- [#5526](https://github.com/xyflow/xyflow/pull/5526) Add `resizeDirection` prop to
  `ResizeControls`

### Patch Changes

- [#5538](https://github.com/xyflow/xyflow/pull/5538) Suppress
  `svelte/prefer-svelte-reactivity` warnings
- [#5529](https://github.com/xyflow/xyflow/pull/5529) Fix selected set to false on every
  node and edge eagerly
- [#5528](https://github.com/xyflow/xyflow/pull/5528) Let `NodeResizer` props change
  during ongoing resize
- [#5546](https://github.com/xyflow/xyflow/pull/5546) Do not crash mini map if all nodes
  are hidden
- [#5547](https://github.com/xyflow/xyflow/pull/5547) Always call `onMoveEnd` when
  `onMoveStart` was called
