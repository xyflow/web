# useNodeConnections()

[Source on GitHub](https://github.com/xyflow/xyflow/blob/main/packages/svelte/src/lib/hooks/useNodeConnections.ts)

This hook returns an array of connections on a specific node, handle type ('source', 'target') or handle ID.

```svelte
<script>
  import { useNodeConnections } from '@xyflow/svelte';
  const connections = useNodeConnections({ handleType: 'target', handleId: 'my-handle' });
</script>

<div>There are currently {connections.length} incoming connections!</div>
```
