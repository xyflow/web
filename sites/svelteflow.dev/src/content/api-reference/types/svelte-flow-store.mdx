# SvelteFlowStore

The `SvelteFlowStore` type is the structure of the internal Svelte Flow Store, that you can access via the [useStore](/api-reference/hooks/use-store) hook.

<Callout type="info">
  The internal Svelte Flow store should only be used for advanced use cases.
  It's not recommended to be used directly.
</Callout>

```ts

  nodes: NodesStore;

  edgeLookup: Readable<EdgeLookup>;

  fitViewOnInit: Writable<boolean>;
  fitViewOnInitDone: Writable<boolean>;
  fitViewOptions: Writable<FitViewOptions>;
```
