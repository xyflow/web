## xyflow | React Flow | Svelte Flow docs

This folder contains the source code of the [xyflow website](https://xyflow.com). It is built with [nextra](https://nextra.site/).

### Installation

```
$ pnpm install
```

### Development

This starts a dev server on http://localhost:3003

```
$ pnpm start
```

### Build

```
$ pnpm build
```
