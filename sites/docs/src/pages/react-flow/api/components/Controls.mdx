# &lt;Controls />

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/react/src/additional-components/Controls/Controls.tsx#L25)

The `<Controls />` component renders a small panel that contain convenient
buttons to zoom in, zoom out, fit the view, and lock the viewport.

```tsx
import ReactFlow, { Controls } from '@xyflow/reactlfow'

export default function Flow() {
  return (
    <ReactFlow nodes={[...]} edges={[...]}>
      <Controls />
    </ReactFlow>
  )
}
```

## Props

The `<Controls />` component accepts any prop valid on a `<div />` element, as
well as the following:

<PropsTable {...controlsProps} />

## Notes

- To extend or customise the controls, you can use the [`<ControlButton />`](./ControlButton)
