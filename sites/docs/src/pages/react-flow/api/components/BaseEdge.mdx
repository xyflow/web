# &lt;BaseEdge />

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/react/src/components/Edges/BaseEdge.tsx#L7)

The `<BaseEdge />` component gets used internally for all the edges. It can be used inside a custom edge and handles the invisible helper edge and the edge label for you.

```js
import { BaseEdge } from '@xyflow/reactflow';

export function CustomEdge({ sourceX, sourceY, targetX, targetY, ...props }) {
  const [edgePath] = getStraightPath({
    sourceX,
    sourceY,
    targetX,
    targetY,
  });

  return <BaseEdge path={edgePath} {...props} />;
}
```

## Notes

- If you want to use an edge marker with the `BaseEdge` component, you need to pass the `markerStart` or `markerEnd` props of `EdgeProps` to the `BaseEdge`.
