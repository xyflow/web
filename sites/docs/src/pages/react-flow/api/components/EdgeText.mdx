# &lt;EdgeText />

[Source](https://github.com/wbkd/react-flow/blob/961d1295b4f056dec6532ff785a3cc1de60dc1f6/packages/react/src/components/Edges/EdgeText.tsx#L7C38-L7C51)

You can use the `<EdgeText />` as a helper component to display text within your custom edges.

```jsx
import { EdgeText } from '@xyflow/reactflow';

export function CustomEdgeLabel({ label }) {
  return (
    <EdgeText
      x={100}
      y={100}
      label={label}
      labelStyle={{ fill: 'white' }}
      labelShowBg
      labelBgStyle={{ fill: 'red' }}
      labelBgPadding={[2, 4]}
      labelBgBorderRadius={2}
    />
  );
}
```
