# Graph Utils

If you're working with a controlled flow (i.e. you're managing the state of the
graph yourself) you will typically set up handlers for the `onNodesChange`,
`onEdgesChange`, and `onConnect` events. The functions documented below make it
easier to work with changes to the graph.

```ts
import ReactFlow, {
  isNode,
  isEdge,
  applyNodeChanges,
  applyEdgeChanges,
  addEdge,
  updateEdge,
  getOutgoers,
  getIncomers,
  getConnectedEdges,
  getTransformForBounds,
  getRectOfNodes,
} from '@xyflow/reactflow';
```

## isEdge

```ts
export function isEdge(item: any) => item is Edge;
```

## isNode

```ts
export function isNode(item: any) => item is Node;
```

## applyNodeChanges

```ts
export function applyNodeChanges(changes: NodeChange[], nodes: Node[]) => Node[]
```

## applyEdgeChanges

```ts
export function applyEdgeChanges(changes: EdgeChange[], edges: Edge[]) => Edge[]
```

## addEdge

```ts
expport function addEdge(edge: Edge, edges: Edge[]): Edge[]
```

## updateEdge

```ts
export function updateEdge(oldEdge: Edge, newConnection: Connection, edges: Edge[]) => Edge[]
```

```ts
export function getOutgoers(node: Node, nodes: Node[], edges: Edge[]) => Node[]
```

## getIncomers

```ts
export function getIncomers(node: Node, nodes: Node[], edges: Edge[]) => Node[]
```
