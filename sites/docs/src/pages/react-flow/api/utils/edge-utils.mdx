# Edge Utils

Implementing custom edges can be a bit of hard work. THe functions documented
below make it easier to work with paths in custom edges. You can see how some of
these functions are used in our [custom edges](/react-flow/examples/edges/custom-edge)
example.

```ts
import ReactFlow, {
  getBezierPath,
  getSimpleBezierPath,
  getSmoothStepPath,
  getStraightPath,
  getMarkerEnd,
} from '@xyflow/reactflow';
```

## getBezierPath

```ts
export function getBezierPath(params: GetBezierPathParams) => [
  path: string,
  labelX: number,
  labelY: number,
  offsetX:number,
  offsetY: number
]
```

Note that the return type of this function is a **named tuple**. If you are working
with JavaScript, you should ignore the labels and treat the return value as an
array.

```ts
type GetBezierPathParams = {
  sourceX: number;
  sourceY: number;
  sourcePosition?: Position;
  targetX: number;
  targetY: number;
  targetPosition?: Position;
  curvature?: number;
};
```
