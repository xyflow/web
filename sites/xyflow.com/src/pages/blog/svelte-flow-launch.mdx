## Features

**Interactivity out-of-the-box:** dragging nodes, zooming, panning, selecting multiple nodes, and adding/removing edges are all built-in.

**Easily customizable:** Svelte Flow nodes are components, so you can create custom nodes any way you want. Svelte Flow is focused on the UI and canvas of your application, you bring the data and layouting libraries.

**Built-in Components:** Background, Minimap, Controls, and Panel components are ready-to-use on install.

## Get running with just a few lines of code

After installing `@xyflow/svelte`, you can make your first Svelte Flow app by copy-pasting this into your Svelte project.

```jsx
<script lang="ts">
  import { writable } from 'svelte/store';
  import {
    SvelteFlow,
    Controls,
    Background,
    MiniMap,
  } from '@xyflow/svelte';

  import '@xyflow/svelte/dist/style.css'

  const nodes = writable([
    {
      id: '1',
      data: { label: 'Hello' },
      position: { x: 0, y: 0 }
    },
    {
      id: '2',
      data: { label: 'World' },
      position: { x: 100, y: 100 }
    }
  ]);

  const edges = writable([
    {
      id: '1-2',
      source: '1',
      target: '2',
    }
  ]);
</script>

<SvelteFlow {nodes} {edges} fitView>
  <Controls />
  <Background />
  <MiniMap />
</SvelteFlow>
```
