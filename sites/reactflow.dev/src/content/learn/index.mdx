# Introduction

React Flow is a library for building node-based applications.
Whether you're creating simple static diagrams, immersive data visualizations, or complex visual editors,
React Flow offers extensive customizability for your nodes and edges, and it comes with handy built-in components to get you up and running quickly.

```bash copy npm2yarn
npm install @xyflow/react
```
