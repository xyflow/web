In this example we are using [Tailwind](https://tailwindcss.com/) for styling the flow.

<Callout type="warning">
  Import the React Flow stylesheet into the base layer so that Tailwind can override the
  default styles.
</Callout>

```css filename="index.css"
@import '@xyflow/react/dist/style.css' layer(base);
@import 'tailwindcss';
```
