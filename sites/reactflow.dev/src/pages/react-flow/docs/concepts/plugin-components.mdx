# Plugin Components

React Flow comes with several additional plugin components. In this guide we show you how to use them. We are using our [previous example code](/docs/concepts/core-concepts) here.

### MiniMap

If your flow gets bigger, you might want to get an overview quickly. For this we have built the [`MiniMap` component](/docs/api/plugin-components/minimap). You can easily add it to your flow by adding it as a children:

import CodeViewer from '@/components/code-viewer';

<CodeViewer
  codePath="api-flows/MiniMap"
  applyStyles={false}
  options={{ editorHeight: 500, editorWidthPercentage: 45, wrapContent: true }}
  additionalFiles={['nodes.js', 'edges.js']}
/>
