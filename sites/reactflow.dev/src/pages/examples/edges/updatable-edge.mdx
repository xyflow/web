An edge is updatable by dragging it to another handle if you are using the [`onEdgeUpdate` handler prop](/api-reference/react-flow#edges). The handler gets called after the edge gets dropped to a new handle. You can use the [`updateEdge`](/api-reference/utils/update-edge) helper function to update your edges state accordingly.
