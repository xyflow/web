# &lt;EdgeText />

[Source on GitHub](https://github.com/wbkd/react-flow/blob/f919cb152dd5a92ec3b4d7e4568ffe099bc2af39/packages/react/src/components/Edges/EdgeText.tsx#L7)

You can use the `<EdgeText />` as a helper component to display text within your
custom edges.

```jsx
import { EdgeText } from '@xyflow/react';

export function CustomEdgeLabel({ label }) {
  return (
    <EdgeText
      x={100}
      y={100}
      label={label}
      labelStyle={{ fill: 'white' }}
      labelShowBg
      labelBgStyle={{ fill: 'red' }}
      labelBgPadding={[2, 4]}
      labelBgBorderRadius={2}
    />
  );
}
```

## Props

For TypeScript users, the props type for the `<EdgeText />` component is exported
as `EdgeTextProps`.

<PropsTable {...edgeTextProps} />

Additionally, you may also pass any standard React HTML attributes such as `onClick`,
`className` and so on.
