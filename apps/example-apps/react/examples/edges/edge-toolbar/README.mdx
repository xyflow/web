---
title: Edge Toolbar
description: You can use the EdgeToolbar component to display content on top of an edge.
---

The [`EdgeToolbar` component](/api-reference/components/edge-toolbar) can be used to
display unscaled content on top of an edge. You can use it to display a button at the
center of the edge or a toolbar that floats above the edge for example.
