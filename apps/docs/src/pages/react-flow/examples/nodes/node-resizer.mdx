The [`<NodeResizer />`](/docs/api/nodes/node-resizer) component can be used to add a resize UI for a custom node. The `node-resizer` package also exports a `<NodeResizeControl />` component for implementing a custom resizing UI as shown in this example.

<Callout type="info">

The `NodeResizer` component is **not** part of the `reactflow` package. You need to install it seperately and you need reactflow >= v11.3.3 for this to work.
